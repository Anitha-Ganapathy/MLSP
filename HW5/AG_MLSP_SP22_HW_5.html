Gaussian radius basis function (RBF) kernel¶

RBF kernel that is used to perform nonlinear dimensionality reduction via BF kernel principal component analysis (kPCA).

Implementing the RBF kernel PCA step-by-step

https://sebastianraschka.com/Articles/2014_kernel_pca.html

In order to implement the RBF kernel PCA we just need to consider the following two steps.

  1. Computation of the kernel (similarity) matrix. In this first step, we need to calculate
$$ \kappa\left(\mathbf{x}_{\mathbf{i}}, \mathbf{x}_{\mathbf{j}}\right)=\exp \left(-\gamma\left\|\mathbf{x}_{\mathbf{i}}-\mathbf{x}_{\mathbf{j}}\right\|_{2}^{2}\right) $$


for every pair of points. E.g., if we have a dataset of 100 samples, this step would result in a symmetric 100x100 kernel matrix.

  1. Eigendecomposition of the kernel matrix. Since it is not guaranteed that the kernel matrix is centered, we can apply the following equation to do so:
$$ K^{\prime}=K-\mathbf{1}_{\mathbf{N}} K-K \mathbf{1}_{\mathbf{N}}+\mathbf{1}_{\mathbf{N}} K \mathbf{1}_{\mathbf{N}} $$


where $\mathbf{1}_{\mathbf{N}}$ is (like the kernel matrix) a $N \times N$ matrix with all values equal to $\frac{1}{N}$. [3]

Now, we have to obtain the eigenvectors of the centered kernel matrix that correspond to the largest eigenvalues. Those eigenvectors are the data points already projected onto the respective principal components.

Do kernel PCA on this data set to transform them into a new 3-dimensional feature space, i.e. $$ \boldsymbol{X} \in \mathbb{R}^{2 \times 152} \Rightarrow \text { Kernel PCA } \Rightarrow Z \in \mathbb{R}^{3 \times 152} $$

https://github.com/JAVI897/Kernel-PCA/blob/master/kernels.py
https://odsc.medium.com/implementing-a-kernel-principal-component-analysis-in-python-495f04a7f85f

def rbf_kernel_pca(X, gamma, n_components):
    sq_dists = pdist(X,  metric='sqeuclidean')    

    mat_sq_dists = squareform(sq_dists)      
    K = exp(-gamma * mat_sq_dists)    
    print('K shape: ',K.shape)
    N = K.shape[0]
    one_n = np.ones((N,N)) / N
    K = K - one_n.dot(K) - K.dot(one_n) + one_n.dot(K).dot(one_n)    
    eigvals, eigvecs = eigh(K)
    print('eigvals, eigvecs  shape : ', eigvals.shape, eigvecs.shape )
    eigvals, eigvecs = eigvals[::-1], eigvecs[:, ::-1]    
    print('eigvals, eigvecs  shape : ', eigvals.shape, eigvecs.shape )
    
    X_pc = np.column_stack([eigvecs[:, i] for i in range(n_components)])    
    return X_pc

X_kpca = rbf_kernel_pca(X.T, gamma=450, n_components=3)

def initialize_parameters(n_x, n_h, n_y):
  np.random.seed(99) 

  W1 = np.random.randn(n_h, n_x) * 0.01
  b1 = np.zeros((n_h,1))
  
  W2 = np.random.randn(n_y, n_h) * 0.01
  b2 = np.zeros((n_y,1))

  parameters = {"W1": W1,
              "b1": b1,
              "W2": W2,
              "b2": b2}

  return parameters

def forward_propagation(X, parameters):
  # Retrieve each parameter from the dictionary "parameters"
  W1 = parameters['W1']
  b1 = parameters['b1']
  W2 = parameters['W2']
  b2 = parameters['b2']
  
  # Implement Forward Propagation
  Z1 = np.dot(W1,X) + b1
  A1 = activation(Z1)

  Z2 = np.dot(W2,A1) + b2
  A2 = activation(Z2)
  
  cache = {"Z1": Z1,
            "A1": A1,
            "Z2": Z2,
            "A2": A2}
  # print(cache)
  return A2, cache

def compute_loss(A2, Y):
  m = Y.shape[1] # number of example

  logprobs = np.dot(Y,np.log(A2).T) + np.dot((1-Y),np.log((1-A2)).T)
  loss = -logprobs/m
  loss = float(np.squeeze(loss))  
  
  return loss

def backward_propagation(parameters, cache, X, Y):
  m = X.shape[1]
    
  # First, retrieve W1 and W2 from the dictionary "parameters".
  W1 = parameters['W1']
  W2 = parameters['W2']

  # Retrieve also A1 and A2 from dictionary "cache".
  A1 = cache['A1']
  A2 = cache['A2']

  # Backward propagation: calculate dW1, db1, dW2, db2. 
  dZ2 = A2 - Y
  dW2 = (1/m) * np.dot(dZ2,A1.T)
  db2 = (1/m) * np.sum(dZ2,axis=1, keepdims=True)
  dZ1 = np.dot(W2.T, dZ2) * (1 - np.power(A1, 2))
  dW1 = (1/m) * np.dot(dZ1, X.T)
  db1 = (1/m) * np.sum(dZ1, axis=1, keepdims=True)

  grads = {"dW1": dW1,
            "db1": db1,
            "dW2": dW2,
            "db2": db2}
  
  return grads

def update_parameters(parameters, grads, lr):
    # Retrieve each parameter from the dictionary "parameters"
    W1 = parameters['W1']
    b1 = parameters['b1']
    W2 = parameters['W2']
    b2 = parameters['b2']

    # Retrieve each gradient from the dictionary "grads"
    dW1 = grads['dW1']
    db1 = grads['db1']
    dW2 = grads['dW2']
    db2 = grads['db2']

    # Update rule for each parameter
    W1 = W1 - lr*dW1
    b1 = b1 - lr*db1
    W2 = W2 - lr*dW2
    b2 = b2 - lr*db2

    parameters = {"W1": W1,
                  "b1": b1,
                  "W2": W2,
                  "b2": b2}
    
    return parameters

def nn_model(X, Y, n_h, num_iterations = 10000, print_loss=False):
    lr = 0.0005
    np.random.seed(42)
    n_x, n_h, n_y = layer_sizes(X, Y, n_h=n_h)
    loss_calc = []
    # loss = 0    
    # Initialize parameters
    parameters = initialize_parameters(n_x, n_h, n_y)

    # Loop (gradient descent)
    for i in range(0, num_iterations):
         
        # Forward propagation. Inputs: "X, parameters". Outputs: "A2, cache".
        A2, cache = forward_propagation(X, parameters)
        
        # loss function. Inputs: "A2, Y, parameters". Outputs: "loss".
        loss = compute_loss(A2, Y)
        loss_calc.append(loss)
        # Backpropagation. Inputs: "parameters, cache, X, Y". Outputs: "grads".
        grads = backward_propagation(parameters, cache, X, Y)
 
        # Gradient descent parameter update. Inputs: "parameters, grads". Outputs: "parameters".
        parameters = update_parameters(parameters, grads, lr)

        # Print the loss every 1000 iterations
        if print_loss and i % 1000 == 0:
            print (f"Loss after iteration {i}: {loss}")
    print ("\nFinal Loss after training: ", loss)
    return parameters, loss_calc

First, for digit class $C$, let's convert each of the 45 utterances into an MFCC spectrogram. You will use a toolbox for this: librosa.feature.mfcc. As a result, you produce $\boldsymbol{X}_{i}^{(C)} \in$ $\mathbb{R}^{20 \times T_{i}^{(C)}}$, where $T_{i}^{(C)}$ is the number of MFCC vectors of the $i$-th utterance in the digit class $C$. Store both the MFCC matrix $\boldsymbol{X}_{i}^{(C)}$ and length information $T_{i}^{(C)}$. Repeat this for all 45 utterances.

def audio_to_mfcc(file_name):
  y, sr = librosa.load(file_name , sr = None)
  mfcc_output =  librosa.feature.mfcc(y=y, sr=sr)
  # print("mfcc_output shape : ", mfcc_output.shape)
  return mfcc_output
